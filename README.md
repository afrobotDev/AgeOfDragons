# Age of Dragons
Age of Dragons is a game about resource management and strategy

## Installation
How to get the environment running:
\`\`\`bash
git clone https://github.com/YitbarekEnd/age_of_dragons
cd age_of_dragons
pip install -r requirements.txt
\`\`\`

## Usage
How to run the project:
\`\`\`
python main.py
\`\`\`
